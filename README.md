# 🏓 Gala Table Tennis - Electrified

An arcade-style, high-voltage table tennis experience built with pure JavaScript and Web Audio API.

### 🎮 How to Play
* **Move:** Use **A / D** keys or **Left / Right Arrows**.
* **Pause:** Press **Spacebar** to freeze the action.
* **Music:** Use the toggle button in the UI to enable/disable the electrified beat.

### 🏆 Rules
* **Rule of Eleven:** First to 11 points wins (must win by 2).
* **Service:** Service changeovers occur every 2 points.
* **Match Timer:** You have 90 seconds to secure your victory!
